開発物詳細

腎臓の構造を利用した深層学習によるCT画像中の腎臓がんのセグメンテーション

Github : https://github.com/t-rryyyoooo/aligned

言語 : Python / フレームワーク : Keras, Pytorch

担当：環境構築、モデル構築、画像への前処理、後処理

セグメンテーションとは、ピクセル（3次元画像の場合はボクセル ）ひとつひとつに対して、ラベルを付与することで、対象の種類や位置、形までをも認識する技術のことをいいます。また、ディープラーニング とは多層のニューラルネットワークを用いることで表現力、学習能力を高めた機械学習の手法の1つを指します。ディープラーニングを用いた手法が2015年に細胞のセグメンテーションにおいて高精度を叩き出したことで、現在では様々な対象で応用されています。また、腎臓は、内部に円錐状の腎錐体と呼ばれるものを10個前後持つため特有な構造を持ちます。私は、他臓器にはない腎臓特有の構造を利用することで腎臓がんのセグメンテーション精度を向上させる研究しています。




筋トレ管理アプリ

Github : https://github.com/t-rryyyoooo/app

URL: http://153.127.46.157

言語 : HTML/CSS, Python / webフレームワーク : Django

担当：1人での開発でしたので、全て自分で行いました。

行った筋トレの種目や重量、セット数などを記録するアプリです。記録を残す機能を始め、ログイン機能、種目毎、日付毎の一覧表示、カレンダーによる記録の概要表示、カレンダー上の日付をクリックすると、詳細記録を表示できる機能を実装しました。




Atcoderコンテストにおけるサンプルの自動抽出とコードの自動提出

Github : https://github.com/t-rryyyoooo/atcoder

言語 : Python

Atcoderとは競技プログラミングサイトのことです。ここで出題される各問題にそれぞれに付随するコードの正当性を確認するためのサンプルを自動抽出するプログラムを作成しました。また、抽出したサンプル全てに正解した場合に自動的に書いたコードをAtcoder上に提出するプログラムも作成しました。
